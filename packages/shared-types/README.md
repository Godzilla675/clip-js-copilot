# @ai-video-editor/shared-types

This package contains shared TypeScript type definitions for the AI Video Editor project.
It is designed to be a dependency for both the frontend (web) and backend services.

## Usage

```typescript
import { Project, Timeline, Asset } from '@ai-video-editor/shared-types';
```

## Types

- **Project**: Core project structure.
- **Timeline**: Timeline, Track, Clip definitions.
- **Asset**: Asset management types.
- **Transcript**: Audio transcription types.
- **MCP Tools**: Return types for MCP tools.
- **LLM**: LLM provider configurations.
